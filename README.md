## 🚀 طريقة التشغيل

### 1. تثبيت المتطلبات
```bash
npm install
```

### 2. تشغيل البوت
```bash
npm start
```

### 3. ربط الواتساب
- سيظهر QR Code في الطرفية
- افتح واتساب → الأجهزة المرتبطة → ربط جهاز
- امسح الكود

## ⚙️ الميزات التلقائية

- **الترحيب التلقائي:** ترحيب بكل عضو جديد يدخل المجموعة (باللهجة اليمنية)
- **وداع تلقائي:** رسالة وداع لكل عضو يغادر
- **رفض المكالمات:** رفض تلقائي لأي مكالمة يتلقاها البوت
- **حماية من السبام:** منع التكرار المفرط للرسائل
- **الذكاء الاصطناعي:** الرد عند مناداة البوت باسمه "سعيد"
